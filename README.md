# ModelLayer @geovanirangel

O ModelLayer é um componente que abstrai operações de CRUD (Select, Insert, Update e Delete) no seu banco de dados. Baseado em PDO, testado com MySQl.

---

## Instalação

via Composer:

```json
"geovanirangel/modellayer": "^3.0"
```
